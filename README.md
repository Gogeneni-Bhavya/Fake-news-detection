# Fake News Detection System

## Project Description

Fake News Detection System is a Machine Learning project developed using Python.  
This project detects whether a news article is:

- Fake News
or
- Real News

The system uses Natural Language Processing (NLP) and Machine Learning techniques to analyze news text and classify it correctly.

---

## Features

- Reads news dataset from CSV file
- Converts text data into numerical format
- Trains Machine Learning model
- Predicts fake or real news
- Calculates prediction accuracy
- Accepts custom news text from user
- Detects fake news instantly

---

## Technologies Used

- Python
- Pandas
- Scikit-learn
- NLP (Natural Language Processing)

---

## Tools Used

- VS Code / PyCharm
- Python Interpreter
- Command Prompt / Terminal

---

## Dataset Used

- `news.csv`

Dataset contains:
- text → News content
- label → Fake or Real
